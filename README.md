# Tinydot Experiment

## Environemnt

* Ubuntu 12.04
* Ruby 2.1.0-p0
* graphviz 2.26.3

## Usage

```sh
sudo aptitude install graphviz
```

Get `tinydot` by `bundle install` command and execute
`tinydot convert sample.tinydot`. `sample.png` will be generated.
